# Python-GUI-Signal-Analysis
基于 PySide6/PyQt 的数字信号处理可视化系统，包含信号波形生成、频谱分析及 GUI 交互界面。
# 数字信号分析与 GUI 交互系统

## 项目简介
本项目是基于 Python 的工程实践，主要实现了以下功能：
- 使用 **PySide6/PyQt** 设计交互式 GUI 界面。
- 实现了数字信号的读取、处理与实时波形展示。
- 集成了 **Matplotlib/NumPy** 进行频域分析。

## 开发环境
- 操作系统: Windows (MinGW64/Git Bash)
- 编辑器: VS Code
- 主要库: PySide6, NumPy, Matplotlib
